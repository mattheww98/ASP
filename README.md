This package is a PyTorch implementation of the absorption spectrum predicting model [TLOpt](https://github.com/usccolumbia/TLOpt) from Dong and Hu, outlined in their paper, ["Inverse design of composite metal oxide optical materials based on deep transfer learning and global optimization"](https://www.sciencedirect.com/science/article/pii/S0927025620306571)
The model is a simple MLP using compositon-based embeddings from the Materials Agnostic Platform for Informatics and Exploration (Magpie) which we get from the [Matminer package](https://www.nature.com/articles/npjcompumats201628). 
The design of this package is largely based on [CrabNet](https://github.com/anthony-wang/CrabNet/tree/master) from [Wang et al.](https://www.nature.com/articles/s41524-021-00545-1).
